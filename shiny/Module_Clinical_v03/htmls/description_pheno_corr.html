This submodule allows detailed and diversified correlation analysis of one or multiple candidate genes with various clinical cohorts, whereby the input defines the output visualization. Adjustments for any combination of parameters (partial correlation) are possible in the Input tab on the left.

Show available output options ▾
Gene Trait Cohort Output
one one one bar chart, table
one multiple one bar chart, table
one one multiple meta-analysis forest plot, table
one multiple all (meta_sc or meta_om) meta-analysis bar chart, table
multiple one one bar chart, table
multiple multiple one heatmap, table
multiple one all (meta_sc or meta_om) meta-analysis bar chart, table
multiple multiple all (meta_sc or meta_om) meta-analysis heatmap, table
