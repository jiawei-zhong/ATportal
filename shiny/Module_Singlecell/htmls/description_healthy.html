Atorvastatin

Stimulation of in vitro differentiated adipocytes (SGBS) with Atorvastatin. RNA sequencing data is available for three Atorvastatin treated replicates (1 uM) and 12 controls. For more information, please see the Details tab. Outputs currently include heatmaps, volcano plots and summary table for one or multiple genes. More information on processing and interpretation can be found under the Interpretation tab.
